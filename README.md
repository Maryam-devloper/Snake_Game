## 🐍 Snake Game
A classic Snake Game built using **HTML, CSS, and JavaScript**. Control the snake, collect food, and grow as long as possible while avoiding collisions with the walls and your own tail. The game features smooth controls, score tracking, and responsive gameplay for an enjoyable user experience.
### Features
* Classic Snake Game mechanics
* Real-time score tracking
* Keyboard controls for movement
* Game Over detection
* Responsive and user-friendly interface
* Lightweight and fast performance
* Sound effects and animations
### Technologies Used
* **HTML5** – Structure of the game
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Game logic and interactivity
### How to Play
1. Use the **Arrow Keys** to control the snake.
2. Eat the food to increase your score and snake length.
3. Avoid hitting the walls or the snake's own body.
4. Try to achieve the highest score possible!
### 📂 Project Purpose
This project was developed to practice and improve front-end web development skills, including DOM manipulation, event handling, game logic implementation, and responsive design using HTML, CSS, and JavaScript.
